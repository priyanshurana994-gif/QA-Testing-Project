# QA Testing Project

## Project Overview
This project demonstrates manual and API testing skills using a sample e-commerce website.

## Scope
- Login Testing
- Cart Testing
- Checkout Testing
- API Testing using Postman

## Tools Used
- Jira (Bug Tracking)
- Postman (API Testing)
- Excel (Test Cases & Bug Reports)

## Key Highlights
- Created 30+ test cases
- Identified multiple bugs with severity & priority
- Performed API testing with status validation
